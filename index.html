selectable-button.js

A jQuery plugin for turning a <select> DOM element into a selectable button, where the button's content is the selected <option>'s content, and clicking on the button shows a dropdown of all non-selected <options>s.

This plugin generates HTML intended to be styled by Twitter Bootstrap's button dropdown menus.

Demo

Usage

Include selectable-button.js on the HTML page after jQuery (don't forget to include Twitter Bootstrap as well if you aren't going to do your own styling):

<script src="jquery.js"></script>
<script src="selectable-button.js"></script>

Given the following <select> DOM element:

<select>
  <option>Choice 1</option>
  <option>Choice 2</option>
  <option>Choice 3</option>
  <option>Choice 4</option>
</select>

Create a selectable button with the following call:

$('select').selectableButton();
